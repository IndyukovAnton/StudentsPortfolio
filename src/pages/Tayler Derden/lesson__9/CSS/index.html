Селектор

это шаблоны, которые используются для привязки стилевых свойств к элементам в документе.



Background-color

Background-color отвечает за смену заднего фона в определенном теге(например div)



width и height

width отвечает за ширину определенном теге
height отвечает за высоту определенного тега



Стили

Существует три способа подключения CSS к документу HTML: Внутриэлементный или строчный. Стиль задаётся непосредственно внутри определения HTML-тега при помощи атрибута style. Внутридокументный или встроенный. Используется для определения стилей элементов целой веб-страницы, если эта страница имеет свои уникальные стили. Для этого в теге head определяется тег style, в котором задаются все стили для данной веб-страницы. Внешний. Используется для подключения одновременно ко многим страницам сайта. Благодаря этому, изменив всего один файл CSS, можно изменить внешний вид всего сайта.


Свойства

Вот некоторые свойства в CSS:

Background: задаёт фон элемента.

Border: задаёт границы элемента.

Border-radius: задаёт скругление рамок.

Margin: задаёт отступы от других элементов.

Padding: задаёт внутренние отступы.

Также существуют свойства, применяемые к тексту:

Color: задаёт цвет текста.

Font-weight: делает текст жирным.

Text-align: задаёт выравнивание текста по центру.

Text-shadow: добавляет тень тексту.
