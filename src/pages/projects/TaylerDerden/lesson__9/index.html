Тег a

Этот тег являестя самозакрывающимся
В тег а можно вставить ссылку на сайт, на который можно перейти при нажатиии на текст


Пример: < a href="ссылка"> текст, при нажатии которого человек перейдет по ссылке < /a>

Как это выглядит: текст, при нажатии которого человек перейдет по ссылке

Тег div является закрывающемся
Тег div это секция в которую можно класть другие теги

Теги b, em, i, ins, mark, strong

они являются закрывающемся
b и strong делают текст жирным.
mark выделяет текст желтым
Тег em выделяет текст курсивом
Тег i используется для изменения голоса или настроения части текста, чтобы передать его другое качество: официальное название, идиома из другого языка, транслитерация или ремарки
Тег ins предназначен для выделения текста, который был добавлен в новую версию документа.
Что бы использовать эти теги, просто поместите между ними текст



Тег span

строчный элемент, который используется для стилизации части текста. Например, с его помощью можно поменять цвет, размер шрифта или начертание, добавить и убрать подчёркивание.

< span> Ваш текст< /span>

Как выглядит:
Ваш текст



Теги ol, li, ul

Тег ol и ul создают списки. ol создает нумерованный список, ul для маркированного
li это значение/пункт в этих списках

Примеры:
ul

  • 1

ol
  1. 1


Как пишется (для обоих случаев оно одинаковое):
< ol> < li>Ваш пункт< /li> < /ol>


Теги select, option

Тег select представляет собой элемент управления, который позволяет создать раскрывающийся (выпадающий) список.
option определяет пункты раскрывающегося списка (параметры для выбора), он применяется как вложенный элемент тега select.


Пример: < select name="Ваше название" id="ваш ID"> < option value="Ваше значение">Текст который будет отображаться< /option> < /select>


Тег iframe

Тег iframe встраивает на веб-страницу документы, видео и интерактивные медиафайлы и прочие части содержимого из других источников.


Пример: < iframe src="Ссылка на документ" frameborder="0">Ваш текст< /iframe>

тег link

link жто ссылка на внешний Ресурс определяет отношения между текущим документом и внешним ресурсом. Этот элемент чаще всего используется для ссылки на stylesheets, а также для создания иконок сайта (как для иконок в стиле "favicon", так и для иконок домашних экранов и приложений мобильных устройств) среди прочего.



тег meta

тег meta предназначен для предоставления структурированных метаданных (дополнительных, сопроводительных) о веб-странице. Как правило, указываются в заголовке. Элемент meta принимает как минимум четыре атрибута: content, http-equiv, name и scheme.



Симантические теги

Семантические HTML-элементы — это теги, которые описывают содержащийся в них контент. Они предоставляют информацию о контенте, которая может помочь поисковым системам понять структуру и значение веб-страницы. Использование семантических HTML-элементов также делает код более читаемым и поддерживаемым, поскольку в нем четко указано назначение каждого элемента.
