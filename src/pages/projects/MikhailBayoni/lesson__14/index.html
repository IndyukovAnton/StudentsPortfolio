CSS Properties Cheat Sheet

1. Color and Background:

color: Sets the text color of an element.

color: blue;

background-color: Sets the background color of an element.

background-color: lightgray;

background-image: Sets an image as the background of an element.

background-image: url('image.jpg');

background-repeat: Specifies if/how a background image is repeated.

background-repeat: no-repeat;

2. Text:

font-size: Sets the size of the font.

font-size: 16px;

font-family: Specifies the font for the text.

font-family: Arial, sans-serif;

font-weight: Sets the weight (boldness) of the font.

font-weight: bold;

text-align: Specifies the horizontal alignment of text.

text-align: center;

3. Positioning:

position: Specifies the type of positioning for an element (static, relative, absolute, fixed, sticky).

position: relative;

top, right, bottom, left: Specifies the offset from the respective edge of the containing element.

top: 10px;
right: 20px;
